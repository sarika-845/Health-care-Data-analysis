## üìä Data Quality & Assumptions

- Approximately "15% of the dataset contains unknown or missing values" in certain columns.
- These values were intentionally retained as "Unknown" instead of being removed or imputed.
- This decision was made to preserve data integrity and avoid introducing false or misleading information.
- Retaining unknown values helps reflect real-world healthcare data, where incomplete or unrecorded information is common.
- All analyses and visualizations account for the presence of these unknown values.
