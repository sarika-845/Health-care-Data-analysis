# Healthcare Data Analysis (EDA)

This project is a Python script that performs **Exploratory Data Analysis (EDA)** on a healthcare dataset.  
It focuses on cleaning patient data and visualizing relationships between key variables.

## Features
- Data Cleaning: Handle missing values with median imputation and categorical filling
- Univariate Analysis: Age distribution, Medical Condition counts
- Bivariate Analysis:
  - Age vs Blood Pressure (scatter + regression line)
  - Average Age by Medical Condition (barplot)
  - Gender vs Medical Condition (crosstab + heatmap)

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
